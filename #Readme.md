# Temperature  monitoring & LCD Display 
# using Arduino

// Description:
This project is a Temperature Monitoring System built using Arduino. It reads temperature data from a sensor and displays it on a 16x2 LCD display in real time.
The project is developed and managed using VS Code, and the code is uploaded to Arduino for execution.

//Objective
To measure and display real-time temperature using a sensor and LCD display.

//components Required:
Arduino Board (Uno/Nano)
Temperature Sensor (DHT11)
16x2 LCD Display (I2C)
Jumper Wires
Breadboard

//Tools Used:
VS Code
Arduino IDE / Arduino Extension
Git & GitHub

//How It Works
1.The temperature sensor reads the surrounding temperature
2.Arduino processes the sensor data
3.The value is displayed on the 16x2 LCD
4.The display updates continuously

//How to Run in VS Code
1.Open project folder in VS Code
2.Install Arduino Extension
3.Connect Arduino via USB
4.Select correct board and port
5.Upload the code
6.Open Serial Monitor (optional)
7.Check output on LCD

//Applications:
Room temperature monitoring
Weather station projects
Embedded system learning
